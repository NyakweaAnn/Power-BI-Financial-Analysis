# 📊 Financial Sales Analysis Dashboard
**Project Date:** April 1, 2026  
**Tools Used:** Power BI Desktop, Power Query, DAX

## 🎯 Project Objective
To transform raw sales data into a professional financial intelligence tool. I focused on ensuring data integrity and reconcilable profit margins.

## 🛠️ Technical Implementation 
* **Data Modeling:** Developed a Star Schema by linking `FactSale` to `DimEmployee`.
* **Relationship Management:** Established **One-to-Many ($1:*$)** cardinality using `Salesperson Key`.
* **Interactive UI:** Built dynamic **Slicers** to allow stakeholders to filter by Employee and Year.

## 📈 Key Achievements
* **Profit Tracking:** Successfully calculated a total portfolio profit of **$1.09M**.
* **Visual Hierarchy:** Organized the report with high-level **KPI Cards** at the top and granular **Audit Tables** at the base.
* **Drill-Down Capability:** Enabled cross-filtering between bar charts and tables for detailed transaction audits.
